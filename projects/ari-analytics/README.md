# ari-analytics

Custom analytics dashboard for [Arab Reform Initiative](https://www.arab-reform.net).

Combines Google Analytics 4, Google Search Console, Zoom webinar, and Mailjet newsletter data with WordPress content metadata into a multi-page React dashboard protected by Google OAuth2 login.
